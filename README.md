# Como executar

Para windows executar

```
C:\FlutterProjects\Otilio-Flutter\run_web.ps1
```

Para linux executar

```
C:\FlutterProjects\Otilio-Flutter\run_web.sh
```

Adicionar chave de api do google maps em .env na raiz do projeto com a chave: GOOGLE_MAPS_API_KEY
